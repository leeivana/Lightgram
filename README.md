# Lightgram
Lightgram is IOS messaging app which uses end to end encryption for messages.

# Tech Stack
<ul>
  <li>
    React-Native
  </li>
  <li>
    MobX
  </li>
  <li>
    Amazon Web Services
    <ul>
      <li>
        AWS AppSync (GraphQL API)
      </li>
      <li>
        AWS Cognito
      </li>
      <li>
        AWS DynamoDB
      </li>
      <li>
      Simple Notification Services
      </li>
    </ul>
  </li>
  <li>
    Virgil
  </li>
</ul>
